    Tableau Hacks

    Tableau 实用技巧整理，覆盖日常 Dashboard 搭建中最常遇到的场景。

    时间范围过滤器（Time Range Filter）

    大多数 Dashboard 都需要一个时间范围选择器，实现步骤：

    1. 创建 Start Date 和 End Date 两个参数（Parameter），类型选 Date
    2. 新建计算字段：[Date] >= [Start Date] AND [Date] <= [End Date]
    3. 将该计算字段拖到 Filters，选择 True
    4. 右键两个参数 → Show Parameter Control，展示在 Dashboard 上

    Container

    使用 Horizontal / Vertical Container 来组织 Dashboard 布局，比直接拖拽 Sheet 更容易对齐和调整大小。

    LOD（Level of Detail）表达式

    LOD 表达式让你可以在不同粒度上计算聚合，不受视图维度的影响。

    // FIXED：按 Customer 计算总销售额，忽略视图维度
{ FIXED [Customer Name] : SUM([Sales]) }

// INCLUDE：在视图粒度基础上增加维度
{ INCLUDE [Order ID] : SUM([Sales]) }

// EXCLUDE：从视图粒度中排除某个维度
{ EXCLUDE [Region] : SUM([Sales]) }
