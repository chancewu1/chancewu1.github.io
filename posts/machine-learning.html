    机器学习

    Supervised Machine Learning

    监督学习是指从有标注的训练数据中学习一个函数，该函数可以用来预测新数据的标签。常见算法包括线性回归、逻辑回归、支持向量机、决策树和神经网络。

    Unsupervised Machine Learning

    无监督学习是指在没有标注数据的情况下，从数据中发现隐藏的模式或结构。常见算法包括 K-Means 聚类、主成分分析（PCA）和自编码器。
