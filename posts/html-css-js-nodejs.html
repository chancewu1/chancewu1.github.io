    一文讲清HTML, CSS, JavaScript, Node.JS

    用最简单的方式理解前端三件套和 Node.js 的关系。

    三层结构

    • HTML — 骨架，描述网页的结构和内容。
    • CSS — 皮肤，描述网页的样式和外观。
    • JavaScript — 灵魂，赋予网页行为和交互。

    三者合一才是一个完整的人，缺少任何一个就是"植物人"状态。

    发展历史

    • HTML：1993 年草案 → HTML 4.01 (1999) → HTML5 (2014)
    • CSS：CSS1 (1996) → CSS2.1 → CSS3（模块化，持续演进）
    • JavaScript：Brendan Eich 用 10 天发明（1995），ECMAScript 标准化（1997）

    Node.js 是什么

    Node.js 是基于 Google V8 引擎构建的运行时环境，让 JavaScript 可以在浏览器之外运行，用于服务端开发。

    // 浏览器中的 JS：操作 DOM
document.getElementById("title").textContent = "Hello";

// Node.js 中的 JS：操作文件系统
const fs = require("fs");
fs.writeFileSync("hello.txt", "Hello from Node!");
