    3 Types of products and the key metrics ▸

    1. E-Commerce

    • Key Products: Amazon, eBay, Shopify, WooCommerce.
    • Key Metrics:
    • Conversion Rate: Percentage of visitors who make a purchase.
    • Average Order Value (AOV): Revenue per transaction.
    • Customer Lifetime Value (CLTV): Revenue generated by a customer over their lifetime.
    • Cart Abandonment Rate: Percentage of initiated checkouts not completed.
    • Gross Merchandise Value (GMV): Total sales volume.
    • Return on Ad Spend (ROAS): Revenue generated per advertising dollar spent.2. Travel and Hospitality
    • Key Products: Booking.com, Airbnb, Skyscanner, Uber.
    • Key Metrics:
    • Booking Volume: Number of bookings made.
    • Revenue per Booking: Average revenue generated from each booking.
    • Occupancy Rate: Percentage of available properties or seats booked.
    • Customer Satisfaction Score (CSAT): Guest feedback.
    • Cancellation Rate: Percentage of bookings canceled.
    • Time to Book: Average time from search to booking completion.3. Healthcare and Wellness
    • Key Products: Teladoc, Fitbit, BetterHelp, Epic.
    • Key Metrics:
    • Appointment Completion Rate: Percentage of scheduled appointments attended.
    • Patient Retention Rate: Percentage of repeat patients.
    • Monthly Active Users (MAU): Number of unique active users per month.
    • Health Improvement Metrics: User-reported outcomes (e.g., steps, weight loss).
    • Time to Resolution: Time taken to resolve a patient’s issue or provide a diagnosis.
    • Revenue per Consultation: Average revenue per patient session.4. Social Media and Communication
    • Key Products: Facebook, Slack, WhatsApp, Zoom.
    • Key Metrics:
    • Monthly Active Users (MAU) / Daily Active Users (DAU): User engagement levels.
    • Average Session Duration: Time users spend on the platform.
    • Retention Rate: Percentage of users returning over time.
    • Virality Coefficient: Rate at which users invite others.
    • Churn Rate: Percentage of users leaving the platform.
    • Revenue per User (ARPU): Average revenue generated per user.5. Education and E-Learning
    • Key Products: Coursera, Khan Academy, Duolingo.
    • Key Metrics:
    • Course Completion Rate: Percentage of users completing a course.
    • Engagement Rate: Frequency and duration of user activity.
    • Active Learners: Number of users actively learning per period.
    • Revenue per Learner: Average income generated per user.
    • Certification Rate: Percentage of users earning certificates.
    • Content Rating Score: Feedback on course quality.6. Financial Services
    • Key Products: Mint, Robinhood, PayPal, Stripe.
    • Key Metrics:
    • Total Transaction Value (TTV): Volume of money processed.
    • Revenue per Transaction: Average revenue generated per financial interaction.
    • Active Accounts: Number of users actively using services.
    • Net Promoter Score (NPS): Customer satisfaction and loyalty.
    • Fraud Rate: Percentage of fraudulent transactions.
    • Retention Rate: Percentage of users retained month-over-month.7. Real Estate
    • Key Products: Zillow, Realtor.com, Redfin.
    • Key Metrics:
    • Listing Engagement: Views, clicks, and inquiries per property.
    • Leads Generated: Number of interested buyers/renters.
    • Conversion Rate: Percentage of leads converting to sales.
    • Average Property Value: Value of listed properties.
    • Time on Market: Average time a listing remains unsold.
    • Commission per Sale: Revenue generated per transaction.8. Entertainment and Media
    • Key Products: Netflix, TikTok, Steam.
    • Key Metrics:
    • Monthly Active Users (MAU): Number of active users in a month.
    • Average Viewing Time: Time spent consuming content.
    • Subscriber Retention Rate: Percentage of subscribers retained.
    • Content Engagement Rate: Likes, shares, comments on media.
    • Revenue per Subscriber: Average revenue generated per user.
    • Content Production ROI: Revenue generated relative to production costs.9. Retail and Consumer Services
    • Key Products: DoorDash, Square POS, Toast.
    • Key Metrics:
    • Delivery Completion Rate: Percentage of successful deliveries.
    • Revenue per Delivery: Average income from each order.
    • Order Accuracy Rate: Percentage of correctly fulfilled orders.
    • Average Transaction Value: Revenue per sale in retail.
    • Customer Retention Rate: Percentage of repeat customers.
    • Operational Efficiency: Time per transaction or delivery.10. Logistics and Transportation
    • Key Products: FedEx Ship Manager, Route4Me.
    • Key Metrics:
    • Delivery Timeliness: Percentage of on-time deliveries.
    • Cost per Delivery: Expense incurred per shipment.
    • Fleet Utilization Rate: Percentage of active vehicles.
    • Route Optimization Score: Efficiency of delivery routes.
    • Package Accuracy Rate: Percentage of undamaged and correct deliveries.
    • Carbon Footprint Reduction: Environmental impact metrics.11. Business Services
    • Key Products: Salesforce, Jira, Workday.
    • Key Metrics:
    • Sales Pipeline Velocity: Time taken to close deals.
    • Task Completion Rate: Percentage of completed tasks.
    • Employee Satisfaction: Measured through surveys or ratings.
    • Customer Churn Rate: Percentage of lost customers.
    • Revenue per Employee: Average income generated per staff member.
    • SLA Compliance Rate: Percentage of tasks meeting service level agreements.12. Creative and Design
    • Key Products: Canva, Adobe Creative Suite.
    • Key Metrics:
    • User Adoption Rate: Percentage of new users joining and actively using.
    • Time to Task Completion: Time users spend completing tasks.
    • Design Engagement Rate: Likes, shares, and downloads of created content.
    • Subscription Retention Rate: Percentage of users renewing subscriptions.
    • Average Revenue per User (ARPU): Income per user.
    • Content Creation Volume: Total projects or designs created.13. Data and Analytics
    • Key Products: Tableau, Snowflake, Google Analytics.
    • Key Metrics:
    • Query Response Time: Speed of data retrieval.
    • Dashboard Engagement: Views and interactions with visualizations.
    • User Retention: Percentage of repeat users.
    • Data Accuracy Rate: Percentage of error-free data insights.
    • Revenue per Insight: Income generated by actionable analytics.
    • Time to Insight: Average time to generate meaningful results.14. Community and Events
    • Key Products: Eventbrite, Discord, Meetup.
    • Key Metrics:
    • Event Attendance Rate: Percentage of registered users attending.
    • Engagement Rate: Interactions within the platform (comments, RSVPs).
    • Revenue per Event: Income generated per hosted event.
    • Community Growth Rate: Percentage increase in members.
    • Active Members: Number of consistently engaged users.
    • Sponsorship Revenue: Income from event or community sponsorships.
